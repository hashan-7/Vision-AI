### ✨ **Project Overview**
**Visionary AI** is a premium image generation dashboard built for creators. It leverages distributed neural clusters to turn high-fidelity text prompts into cinematic masterpieces. Featuring a modern **Glassmorphism UI**, secure **OAuth2** authentication, and a robust credit management system.

### 🛠 **Key Features**
* 🎨 **Neural Generation:** High-resolution text-to-image synthesis.
* 💎 **Liquid UI:** Ultra-modern dark-themed interface with 25px blur saturation.
* ⚡ **Credit Engine:** Smart daily generation limits (10/day) with 24h auto-reset.
* 🛡️ **OAuth2 Security:** Enterprise-grade Google authentication.
* 🖼️ **Personal Gallery:** High-performance storage and retrieval of neural artifacts.

### ⚙️ **Quick Start**
1.  **Clone the Repo:** `git clone https://github.com/hashan-7/Vision-AI.git`
2.  **Configure:** Rename `application.properties.example` to `application.properties` and add your keys.
3.  **Build & Run:**
    ```bash
    mvn clean install
    mvn spring-boot:run
    ```
