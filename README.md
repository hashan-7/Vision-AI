### 🚀 **Introduction**
**Visionary AI** is a premium image generation dashboard. It turns complex text prompts into cinematic neural artifacts using distributed AI clusters. 

### 🛠 **Tech Stack**
* **Backend:** Spring Boot 3.4, Spring Security (OAuth2).
* **Frontend:** Liquid Glassmorphism UI (HTML5, CSS3, JS).
* **Database:** MySQL (User data & image metadata).
* **AI Engine:** Hosted on **Hugging Face Spaces** using REST API.

### ⚙️ **Setup Instructions**
1.  Clone: `git clone https://github.com/hashan-7/Vision-AI.git`
2.  Configure `application.properties` (Use `application.properties.example` as a template).
3.  Run: `mvn spring-boot:run`
